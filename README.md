## ✨ Sobre o projeto

O site foi desenvolvido com o objetivo de apresentar os serviços da profissional de forma clara, elegante e estratégica, proporcionando uma experiência agradável ao usuário.

Inclui seções como:
- Apresentação profissional
- Serviços oferecidos
- Benefícios do acompanhamento nutricional
- Depoimentos
- Contato
